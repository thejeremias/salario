# Salário

Trata-se de uma aplicação web que apresenta uma listagem de pessoas, incluindo informações adquiridas durante o cálculo de salários.


## Tecnologias

| Tecnologia   | Versão    |
|--------------|-----------|
| Java         | 8         |
| JSF (Mojarra)| 2.2.17    |
| PostgreSQL   | 42.2.23   |
| Hibernate    | 4.3.8     |
| PrimeFaces   | 7.0       |
| JUnit        | 4.13.2    |
